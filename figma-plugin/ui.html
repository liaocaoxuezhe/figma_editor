Getting Started

1 · Connect to Cursor
1
In Cursor, go to Settings → MCP → Add new MCP server.
2
Add this config (or paste into .cursor/mcp.json):
{ "mcpServers": { "figma_editor": { "command": "npx", "args": ["-y", "figma_editor"] } } }
3
Open this plugin in Figma — the status dot turns green when connected. That's it.
You don't need to run anything in the terminal. Cursor starts and manages the server automatically.
2 · Figma Access Token
1
Go to Figma → Settings → Security
2
Click Generate new token. Name it figma_editor and set expiry to 90 days.
3
Enable these scopes:
file_content:read file_metadata:read library_assets:read library_content:read team_library_content:read
4
Copy the token and paste it into the Figma Access Token field in the Settings section.
Your token is stored locally in the plugin and never leaves your machine. It's only used to read your library components.
4 · Start Designing
✦
In Cursor, just chat naturally:
"Design a login screen"
"Build a dashboard using the DS"
"Edit the selected card's color"
figma_editor supports creating frames, shapes, text, SVG icons, auto-layout, effects, variables, and importing design system components — all through chat.
